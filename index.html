Projects

CinePhil

A movie app where user can browse movies using TMDB API and add them to their watch/liked list

  • Can browse up to 10 pages of top rated and upcoming movies
  • Can view detailed description of each movie
  • Can get random movie recommendations after answering 3 simple questions
  • After signing up, user can bookmark or like the movie and see them on their dashboard

Planthood

A simple journaling app to help users keep track of their existing plants that has a built-in search box to directly search from a plant API (trefle API)

  • Can browse plants that are already added to the database
  • Can add new plants and upload photos of the new plants
  • Can display the details of each plant from database
  • (WIP) Trefle API is no longer available; search feature will be replaced with another plant API soon

Hope for Home

A 2-way resource app for the homeless and nonprofits to provide and share local information such food bank, shelter, and career services

  • Can register to the website
  • Can add more specific data to the user database after user logs in for the first time
  • Can display shelter information with embedded Google Maps
